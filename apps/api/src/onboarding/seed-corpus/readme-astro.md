# Astro

**Build the web you want.**

[Astro](https://astro.build) is a website build tool for the modern web — powerful developer experience meets lightweight output.

## Install

The **recommended** way to install the latest version of Astro is by running the command below:

```bash
npm create astro@latest
```

You can also install Astro **manually** by running this command instead:

```bash
npm install astro
```

Looking for help? Start with our [Getting Started](https://docs.astro.build/en/getting-started/) guide.

Looking for quick examples? [Open a starter project](https://astro.new/) right in your browser.

## Official Integrations

Astro ships official integrations for popular frameworks and platforms:

- **UI frameworks**: React, Preact, Solid, Svelte, Vue, Alpine.js
- **Adapters**: Node, Vercel, Cloudflare, Netlify
- **Content**: MDX, RSS, Sitemap
- **Database**: `@astrojs/db`
- **Performance**: Partytown (third-party scripts in a worker)
- **Tooling**: VSCode extension, TypeScript plugin, language server
